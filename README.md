# rgtube

desktop application for watching youtube in zen mode

# features:

## use-case:

rgtube will be an youtube or odysee wrapper that will allow you to watch specific channels without ads and interruptions
you'll make profiles for whatever use case
then with the profiles you'll choose your color scheme (custom or a built in one)
and then you'll choose youtube channels that you want to appear on that profile
then when having selected that profile the app will only display the videos from those channels on a pretty ui
to watch the videos the app will download them using yt-dlp when you click them (it'll prompt you first) and then you can watch it through the app
you can remove the videos by hand or manage them in playlists through the ui
you can also run cleanup scripts (either daily, weekly, or monthly) and exclude specific playlists from the cleanup scripts
very useful

## packaging:

rgtube will eventually be packaged for various linux distributions, windows, and macos
